# Companion Paper Workspace

This directory contains the Markdown-first manuscript scaffold for the MetaboLiteLearner 2.0 companion paper.

## Intended workflow

1. maintain the paper arc in `outline.md`
2. draft section text in numbered section files
3. keep figure notes and TODOs in `notes/`
4. keep manuscript-ready figures under `figures/`
5. keep citation assets under `bibliography/`

## Initial Paper Direction

The paper should expand the original methods framing by arguing that fragmentation structure is not merely a technical representation for metabolomics data, but a biologically meaningful representation from which adaptive metabolic signal can be learned.
